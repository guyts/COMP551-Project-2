For kNN:
1. Run cleandata.py to pre-process text data. 
Input the required file names, ie. train_input.csv, train_output.csv, compact.csv
2. Run knn.py to run the actual prediction. 
Input the required file names, ie. train_compact.csv, test_compact.csv, result.csv
3. The parameters downratio, k, maxFeatures can be changed to vary performance
